了解三角函数

在使用三角函数进行canvas动画的应用之前，复习下高中数学关于三角函数的知识点

  • 一弧度 = 弧长等于半径的弧所对应的圆心角的一弧度
  • 360角度 = 2π （一个圆的弧度）
    • 1弧度 = 180 / π； 1角度 = π / 180
  • 正弦函数：圆上任意一点的y坐标和弧长之间的关联
  • 余弦函数：圆上任意一点的x坐标和弧长之间的关联
  • 三角函数(a是对边，b是领边，c是斜边，θ是角度)
    • sin(θ) = a / c
    • cos(θ) = b / c
    • tan(θ) = a / b
  • 反三角函数
    • arcsin(a/c) = θ
    • arccos(b/c) = θ
    • arctan(a/b) = θ
    • Math.atan2(a,b) = θ

      根据维基百科的解释：已坐标原点为起点，指向（x,y）的射线在坐标平面上与x轴方向之间的角的角度

注意：在Math对象中的三角函数采用的弧度制

具体案例：指南针

Math.atan2(a,b)实现指南针随鼠标旋转

第一步-场景描述：页面上有一个指南针，但鼠标划入画布时，指南针跟着鼠标旋转；同时页面中增加一个按钮， 点击按钮会在画布中随机的位置上产生一个新的指南针，同样也是跟着鼠标旋转

第二步-伪代码分析

html部分
创建一个button按钮，用于点击增加指南针
创建一个canvas元素，用于指南针动画效果展示

js部分
获取canvas对象、与getContext('2d')获取渲染上下文和绘画功能
1：初始化
 - 旋转角度 rotation = 0
 - 存储每次点击新增加的指南针 disks = []
 - 鼠标坐标 point = {}
2：创建画布中的指南针【初始化一个指南针】
 - 创建构造函数，通过let d = new Disk()创建一个已知 坐标和大小的对象d
 - push进disks数组
3：绘制指南针  - 创建函数drawDisk，for循环遍历数组disks中存在的所有指南针对象
 - 计算旋转角度；绘画鼠标小红点、不同颜色三角形组成的指南针
 - 使用requestAnimationFrame调用自身drawDisk
4：添加监听事件  - button元素的click事件，调用构造函数Disk 创建新的指南针
 - canvas元素的mousemove事件， 改变point对象的x与y值，即鼠标的位置更新

知识点总结

canvas.save()的作用：保存画布的状态，后续可以进行旋转、平移、缩放、裁剪操作

canvas.resotore()的作用：恢复上一次使用save之前保存的状态，反正受行旋转、平移、缩放、裁剪影响
